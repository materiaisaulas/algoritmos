#### Coleções

Em Python, **coleções** são estruturas que permitem **agrupar múltiplos valores em uma única variável**, facilitando o armazenamento, o acesso e a manipulação de conjuntos de dados.

Elas são muito úteis para trabalhar com **listas de elementos**, como nomes, números, objetos ou qualquer outro tipo de informação. As coleções permitem organizar dados e realizar operações como percorrer elementos, buscar valores e manipular conjuntos de informações.

As principais coleções integradas em Python são:

| Coleção             | Características                                                         |
| ------------------- | ----------------------------------------------------------------------- |
| **List (`list`)**   | Coleções **ordenadas e mutáveis**, que aceitam **elementos duplicados** |
| **Tuple (`tuple`)** | Semelhantes às listas, porém **imutáveis**                              |
| **Set (`set`)**     | Coleções **não ordenadas** que armazenam **elementos únicos**           |
| **Dict (`dict`)**   | Coleções formadas por **pares chave : valor**                           |

Essas estruturas são extremamente versáteis e são utilizadas em praticamente todos os tipos de programas, desde scripts simples até sistemas mais complexos. Ao longo do aprendizado, compreender **quando utilizar cada tipo de coleção** é essencial para escrever programas mais organizados e eficientes.
