### História e aplicações

A linguagem Python foi criada no final dos anos 1980 por Guido van Rossum, na Holanda. Seu desenvolvimento começou oficialmente em 1989 no Centrum Wiskunde & Informatica (CWI), um centro de pesquisa em matemática e ciência da computação. O nome “Python” foi inspirado no grupo de comédia britânico “Monty Python”, e não na cobra, como muitos pensam. Van Rossum desejava criar uma linguagem que fosse ao mesmo tempo poderosa e fácil de ler, combinando a clareza do pseudocódigo com a funcionalidade de linguagens como C e Perl.

A primeira versão oficial, Python 1.0, foi lançada em 1991, já contendo estruturas importantes como tratamento de exceções, tipos de dados complexos e funções. Ao longo dos anos, a linguagem evoluiu bastante. O Python 2, lançado em 2000, trouxe melhorias significativas, mas tinha limitações de compatibilidade que foram resolvidas com a chegada do Python 3 em 2008 — a versão atualmente recomendada para novos projetos.

Por que Python se tornou tão popular?

Python ganhou destaque por vários motivos:

* Sintaxe simples e legível, ideal para quem está aprendendo a programar.
* Multiplataforma, funcionando em Windows, macOS e Linux.
* Comunidade ativa, com milhares de bibliotecas prontas.
* Versatilidade, podendo ser usado em diversas áreas da computação.

Python é utilizado em diversas áreas do conhecimento, tanto acadêmicas quanto industriais. A seguir, apresentamos alguns exemplos:

:::info
Ciência de Dados e Inteligência Artificial - Python é a principal linguagem usada por cientistas de dados e engenheiros de IA, graças a bibliotecas como pandas, numpy, matplotlib, scikit-learn, tensorflow e keras.

Desenvolvimento Web - Com frameworks como Django e Flask, é possível criar sites completos e robustos de forma rápida.

Automação de Tarefas (Scripts) - Python é excelente para automatizar tarefas do dia a dia, como renomear arquivos, enviar e-mails ou processar planilhas.

Jogos e Gráficos - A biblioteca pygame permite desenvolver jogos simples, e outras bibliotecas como turtle são usadas para ensino de lógica.

Aplicações Acadêmicas e Ensino - Justamente por sua clareza e simplicidade, Python é amplamente usado em cursos introdutórios de programação, em áreas como engenharia, matemática, biologia, economia e até ciências humanas.
:::

# Introdução 

Python é uma linguagem de programação de alto nível, criada no final da década de 1980 por Guido van Rossum, e lançada oficialmente em 1991. Seu principal objetivo sempre foi tornar a programação mais acessível, com uma sintaxe simples, legível e próxima da linguagem humana. Por essa razão, Python é amplamente utilizada tanto por iniciantes quanto por profissionais experientes em diversos campos da computação.

Uma das características que diferencia Python de muitas outras linguagens é sua versatilidade. Ela pode ser utilizada em múltiplos paradigmas de programação, como o paradigma imperativo (procedural), o orientado a objetos e, em certa medida, o funcional. Além disso, por ser uma linguagem interpretada, os programas em Python são executados linha a linha por um interpretador, o que facilita o desenvolvimento incremental e a depuração de código.

Python é uma das linguagens mais populares da atualidade, sendo empregada em uma variedade de áreas. Na ciência de dados, é a linguagem dominante devido ao seu rico ecossistema de bibliotecas como NumPy, Pandas e Matplotlib. Na área de inteligência artificial e aprendizado de máquina, destaca-se com frameworks poderosos como TensorFlow, PyTorch e Scikit-learn. Python também é amplamente utilizada no desenvolvimento web (com frameworks como Django e Flask), automação de tarefas, desenvolvimento de jogos, prototipagem rápida e na área acadêmica, por sua clareza e ampla adoção na comunidade científica.

O aprendizado de Python em um curso universitário traz uma série de benefícios. Por ter uma curva de aprendizado mais suave, é excelente para a introdução à lógica de programação e aos conceitos fundamentais da computação. Além disso, a sua aplicabilidade em áreas diversas permite ao estudante visualizar como os conceitos aprendidos em sala de aula podem ser utilizados na prática, em projetos reais e de pesquisa.

Como exemplo inicial de sua simplicidade, considere o programa abaixo, que exibe uma mensagem na tela:

```python 
print("Hello, World!")

``` 

Esse exemplo, conhecido como “Hello, World!”, é tradicionalmente o primeiro programa escrito ao aprender uma nova linguagem. Em Python, ele é composto por apenas uma linha, o que demonstra o quanto a linguagem pode ser direta e legível.

Nesse trecho de código, temos quatro variáveis: uma string (nome), um número inteiro (idade), um número decimal (altura) e um valor booleano (estudante). Python infere automaticamente o tipo de dado com base no valor atribuído, o que elimina a necessidade de declarações explícitas de tipo.

```python
nome = "Maria"
idade = 20
altura = 1.65
estudante = True
```

Também é possível interagir com o usuário por meio de entrada de dados. No exmplo o código, solicita ao usuário que insira seu nome e sua idade, realiza a conversão da idade para o tipo inteiro (já que a função input() retorna sempre uma string), e imprime uma mensagem personalizada utilizando interpolação de strings.

```python 
nome = "Maria"
idade = 20
altura = 1.65
estudante = True
```
