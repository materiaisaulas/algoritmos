# Flutter para Iniciantes

Este é um exemplo de livro online construído usando **Markdown + GitHub Pages**.

O objetivo deste material é apresentar os conceitos fundamentais do Flutter
e construir um aplicativo simples passo a passo.

## Conteúdo

- Fundamentos do Flutter
- Widgets
- Layout
- Projeto prático
